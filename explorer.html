💻 Emulador de Terminal

O terminal é onde você vive. Cada opção tem características diferentes — GPU acceleration, ligatures, customização e performance.

kitty — zsh

~/projects ❯ node --version
v22.3.0
~/projects ❯ ▊

Kitty

Terminal moderno com GPU acceleration via OpenGL. Suporta imagens inline, tabs e splits nativos, e é altamente scriptável. Favorito da cena ricing.

GPU accelerated imagens inline splits nativos Python config
alacritty
luiz@ubuntu ~ $ cargo build
Compiling ricing v0.1.0
Finished in 2.3s
luiz@ubuntu ~ $ python3 main.py
Server running on :8000
luiz@ubuntu ~ $ ▊

Alacritty

O terminal mais rápido do mundo, escrito em Rust. Zero features desnecessárias — sem tabs, sem splits. Simples, ultra-performático. Config via TOML.

Rust super rápido minimalista TOML config
WezTerm — 1/3 tabs
Tab 1: zsh   Tab 2: node   Tab 3: python
~/projects/ricing on main
❯ git status
nothing to commit
❯ ▊

WezTerm

GPU accelerated, escrito em Rust, configurado em Lua. Multiplexer embutido (tabs, splits, workspaces). O mais completo — compite com tmux sozinho.

Lua config multiplexer GPU accelerated cross-platform

💡 Dica: Qualquer terminal acima pode usar Zsh + Starship, que cuida da aparência do prompt independente do emulador.

🖥️ Customizações GNOME

O que será configurado no seu desktop além do terminal.

🎨

GTK Theme

Tema de janelas e apps GTK. Vamos instalar a versão do tema da paleta escolhida (ex: Catppuccin GTK, Dracula GTK).

🔲

Ícones: Papirus

Pack de ícones flat e moderno, com variantes coloridas para qualquer paleta. Muito popular e ativo.

🧩

Extensions

Blur my Shell — blur no dock e painel
Just Perfection — hide/ajuste de elementos
Dash to Dock — dock moderno

🔤

Fontes

JetBrains Mono Nerd Font para terminal e código. Inter para UI. Suporte completo a ligatures e ícones.

⌨️

Shell: Zsh + Starship

Zsh com Oh My Zsh para plugins (autosuggestions, syntax highlight). Starship para o prompt — bonito e rápido.

⚙️

gsettings

Script para aplicar configurações: tema, ícones, fonte do sistema, botões de janela, e dark mode automaticamente.

⚙️ Dev Stack

Scripts de instalação automatizados para seu ambiente de desenvolvimento.

🟢 Node.js

Instalado via nvm (Node Version Manager). Permite múltiplas versões e atualização fácil.

nvm node LTS npm pnpm

🐍 Python

Instalado via pyenv para gerenciar versões. Inclui pip, venv e poetry para ambientes.

pyenv python 3.12 pip poetry

🦀 Rust

Instalado via rustup — o jeito oficial. Inclui cargo, rustfmt e clippy.

rustup cargo rustfmt clippy

🔧 Ferramentas CLI

Modernizações das ferramentas clássicas do Unix com estética e UX melhoradas.

bat (cat) eza (ls) fd (find) ripgrep fzf zoxide (cd)
